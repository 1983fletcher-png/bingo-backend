# Scene backgrounds (set dressing)

Add WebP/PNG here for real scene backgrounds (~300KB each preferred). Used by themes.css via --pr-scene-bg.

- **studio.webp** — studio / stage backdrop
- **mountains.webp** — mountains / silhouette
- **arcade-carpet.webp** — arcade carpet pattern

If a file is missing, themes.css falls back to gradient-only for that scene (no build break).
