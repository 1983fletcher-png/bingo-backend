# Theme overlays (set dressing)

Add WebP/PNG here for theme texture overlays (~300KB each). Used by themes.css via --stage-overlay.

- **overlay-prestige.webp** — subtle star/noise texture, mostly transparent
- **overlay-retro.webp** — scanline/CRT or subtle film grain
- **overlay-arcade.webp** — neon speckle / arcade glow texture

Optional later: stageframe-prestige.webp, stageframe-retro.webp, stageframe-arcade.webp (transparent frame highlights).
